Logo Usage

Download Logo Files

The Iron Yard is the umbrella brand under which several business units stand. It is vital that the shield mark be consistent throughout any collateral that is produced - print and digital. These rules are meant to be a framework in which to guide designers and developers on the acceptable ways to communicate using The Iron Yard's brand language. We're not saying that we're not flexible - but you have to know the rules before you can break them.

Primary Mark

When in doubt use this version of the logo. This is generally used on white with ample space around it. It can be used on a lighter or darker version of the standard ironyard gray as well - see the header of the website as an example.

On Photography

A common way to use the mark is to position it on top of a photograph. When this occurs, the white version of the shield must me used. This should happen only when layered on top of a duotone feature photo. The white logo can be used when layered on top of solid colors as well - like academy class colors.

Black - a rare use case

This is reserved for 1-color applications where the only ink that is being used is black. From this description, one can assume that no, you won't see this version of the logo on a screen... unless it's a grayscale screen. In which case, kudos for finding one of those.

Color

Download Swatches

The dominant color of The Iron Yard Brand is the standard Gray that is used in the shield mark. Other major colors in the palette denote sub-brands associated with subjects within the Academy. These are featured in the mext section. Each color has hue progressions associated with it with the middle hue being the dominant variation. Practically, these can be used as tone-on-tone variants of the dominant color and used to add variety to any illustration or layout. These hue variants are used to create the duotone imagery seen in the photography section. Sometimes, when creating printed pieces for each of the sub-brands, a darker shield may be used on top of the sub-brand's base hue. See the header of this web page as an example. It is assumed that most print appications will be executed using 1-color or 4-color process plus a spot for the logo if needed. If you really want to get fancy, the duotones can be created using the two spot colors as well, but it's not necessary.

Primary Mark Color

The shield hue is always at the darkest end of the color progressions. The darker color makes the mark more versatile to the desginer as it offers greater contrast when paired with the option of using the one-color while mark.

Do's and Don'ts of Color Usage

Subjects and Subbrands

Download Zip

The Iron Yard Academy offers a variety of different subjects for aspiring students to sink their teeth into. These subjects each have their own unique icons as well as color scheme. This enables each subject to distinguish themselves and their topics easily without straying away from the over arching brand. The icons can be used in both the encapsulating shield as well as without. The unencapsulated icon may be best used when other icons are in the near vicinity as they are on this page. The shield version may best be used when standing alone - for instance on a card or brochure that is solely about that particular subject. These uses should be left up to the designer's discretion. Flexibility is needed with the icons in order to best determine usage, but stronger rules may develop over time. One fast and strong rule is that if the icons are sharing the same page as The Iron Yard shield, the icon should never compete with or crowd the logo.

Typography

Download Fonts

The main typeface that is used throughout The Iron Yard's collateral is Tisa Pro. Flimotype and Gravity are also used, but they are reserved for the construction of the Shield Mark. Tisa Pro Medium is mainly used for headlines and Tisa Pro Regular is used for body copy. The Medium font can be used for body copy as well. Tisa Pro Bold is mainly used for call to action text, or smaller text that needs to attract the viewer's attention. It works very well when displayed in all caps. Using any of the fonts in italics is another useful way to emphasize a word or phrase. The website uses the same typeface with a slightly different an more specific set of rules. Photography

Download Example

Photography plays an important role in The Iron Yard's branding elements, an it can be used to breath some life into the different applications of The Iron Yard brand. The main thing to keep in mind when using imagery is that this is the time that you, the designer, can choose to bring in the human element. What is The Iron Yard without the students, instuctors, professionals, and entrepreneurs that hang around the place? They should be featured, and this is the place to do it. Full color imagery reigns supreme on the website, but in print form, the duotone images rule. The website is vast, and there are many many places to pepper images in and around other brand elements. With standard print collateral, you have one brief shot to get the point across, and the strong duotone imagery helps drive that brand recognition deeper into the viewers skull. That may seem a bit intense... but we're an intense bunch sometimes.
