Welcome to The Iron Yard Academy , Durham’s premier boot camp style code school. We are training developers to fuel the Triangle’s technology economy in North Carolina.

This school represents a unique partnership between a nationally respected code school and an enterprising technology shop, bringing education directly into the market place so that learning and real world experience happen simultaneously.

To connect students directly with potential employers, Academy classes are held in the American Underground, in close proximity to some of the Triangle’s most promising tech start-ups.

Learn to code. Life's too short for the wrong career.

Our intensive coding courses are open in Durham. Study Mobile Engineering, Ruby on Rails or Front End Engineering and launch a new career.

Academy

We offer three intensive Academy courses in Durham. Our grads get jobs with amazing local companies as well as awesome shops around the country.

Learn More

Front End Engineering

Learn Front End Engineering using HTML, CSS and JavaScript and basic server-side tech. At the end of the class you’ll have the skills to create beautiful, fully functional websites and web applications.

Rails Engineering

Learn Ruby on Rails, one of the most popular server-side frameworks in the world. At the end of class you’ll know how to build fast, production-quality full-stack apps.

Python Engineering

Learn to analyze and publish data using Python and machine learning. At the end of the class you’ll make magic happen with real-world data sets and you'll know how to build web apps to show off your wizardry.

City Highlights

Sports, entertainment, the arts – wherever your tastes lead, Durham has it. World class facilities like the Durham Performing Arts Center, the Durham Bulls Athletic Park and Duke University’s Nasher Museum of Art blend with nearly 100 City-owned parks and recreation facilities to make Durham a place where residents Play More! Durham is also recognized as one of the decorated dining destinations in the state with award-winning restaurants and chefs that make an evening out something special.
